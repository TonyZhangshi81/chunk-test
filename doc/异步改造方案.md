
# RAG Chunk Strategy 实验平台 - 异步改造方案

## 1. 改造概述

### 1.1 改造目标

- 将现有同步CLI平台改造为支持高并发的异步API服务
- 保持CLI命令完全可用，用户体验不变
- 支持流式输出，对接前端聊天界面
- 提升系统吞吐量，支持多用户并发请求

### 1.2 改造原则

- **渐进式改造**：优先改造热点路径（检索、LLM调用）
- **向后兼容**：CLI命令100%保持不变
- **最小侵入**：尽量复用现有业务逻辑

### 1.3 技术选型

| 组件 | 技术 | 用途 |
| ------ | ------ | ------ |
| Web框架 | FastAPI 0.115+ | 高性能异步API |
| HTTP客户端 | httpx 0.27+ | 异步HTTP请求 |
| 数据库驱动 | asyncpg + sqlalchemy[asyncio] | 异步数据库访问 |
| 流式输出 | Server-Sent Events (SSE) | 实时流式响应 |
| 任务队列 | Redis + rq (可选) | 长任务异步处理 |

## 2. 架构变更

** 改造前（同步CLI） **

```text
CLI → Service (同步) → Repository (同步) → PostgreSQL/MinIO/API
```

```python
# repositories/chunk_repo.py
from sqlalchemy import select, text
from sqlalchemy.ext.asyncio import AsyncSession

class AsyncChunkRepository:
    def __init__(self, session: AsyncSession):
        self.session = session
    
    async def save(self, chunk: Chunk):
        self.session.add(chunk)
        await self.session.commit()
    
    async def batch_save(self, chunks: List[Chunk]):
        self.session.add_all(chunks)
        await self.session.commit()
    
    async def search_by_vector(
        self, 
        embedding: List[float], 
        dimension: int, 
        document_id: str,
        chunk_type: str,
        embedding_model: str,
        top_k: int = 3
    ) -> List[Chunk]:
        """异步向量检索"""
        # 根据维度选择字段
        vector_field = f"embedding_{dimension}"
        
        # 使用pgvector的余弦距离
        stmt = select(Chunk).where(
            Chunk.document_id == document_id,
            Chunk.chunk_type == chunk_type,
            Chunk.embedding_model == embedding_model,
            getattr(Chunk, vector_field).is_not(None)
        ).order_by(
            text(f"{vector_field} <=> :embedding")
        ).limit(top_k)
        
        result = await self.session.execute(stmt, {"embedding": embedding})
        return result.scalars().all()
    
    async def delete_by_document(self, document_id: str, chunk_type: str):
        await self.session.execute(
            delete(Chunk).where(
                Chunk.document_id == document_id,
                Chunk.chunk_type == chunk_type
            )
        )
        await self.session.commit()

# 保留同步版本供CLI使用
class SyncChunkRepository:
    def __init__(self, session):
        self.session = session
    
    def save(self, chunk: Chunk):
        self.session.add(chunk)
        self.session.commit()
```

```python
# core/embedding_service.py
import httpx
from typing import List, Optional
import asyncio
from tenacity import retry, stop_after_attempt, wait_exponential

class AsyncEmbeddingService:
    def __init__(self, config):
        self.config = config
        self.client = httpx.AsyncClient(
            timeout=httpx.Timeout(120.0),
            limits=httpx.Limits(max_keepalive_connections=20)
        )
    
    @retry(stop=stop_after_attempt(3), wait=wait_exponential(multiplier=1, min=2, max=10))
    async def embed(self, texts: List[str]) -> List[List[float]]:
        """异步批量生成向量"""
        response = await self.client.post(
            f"{self.config.EMBEDDING_API_BASE}/embeddings",
            headers={"Authorization": f"Bearer {self.config.EMBEDDING_API_KEY}"},
            json={
                "model": self.config.EMBEDDING_MODEL,
                "input": texts
            }
        )
        response.raise_for_status()
        data = response.json()
        return [item["embedding"] for item in data["data"]]
    
    async def embed_single(self, text: str) -> List[float]:
        result = await self.embed([text])
        return result[0]
    
    async def close(self):
        await self.client.aclose()

# 同步包装器，供CLI使用
class EmbeddingService:
    def __init__(self, config):
        self.async_service = AsyncEmbeddingService(config)
    
    def embed(self, texts: List[str]) -> List[List[float]]:
        return asyncio.run(self.async_service.embed(texts))
    
    def embed_single(self, text: str) -> List[float]:
        return asyncio.run(self.async_service.embed_single(text))
```

### 3.4 LLM服务异步改造（支持流式）

```python
# core/llm_service.py
import httpx
import json
from typing import AsyncGenerator, List, Optional

class AsyncLLMService:
    def __init__(self, config):
        self.config = config
        self.client = httpx.AsyncClient(timeout=httpx.Timeout(180.0))
    
    async def complete(self, prompt: str) -> str:
        """非流式完成"""
        response = await self.client.post(
            f"{self.config.LLM_API_BASE}/chat/completions",
            headers={"Authorization": f"Bearer {self.config.LLM_API_KEY}"},
            json={
                "model": self.config.LLM_MODEL,
                "messages": [{"role": "user", "content": prompt}],
                "temperature": self.config.LLM_TEMPERATURE,
                "stream": False
            }
        )
        response.raise_for_status()
        data = response.json()
        return data["choices"][0]["message"]["content"]
    
    async def stream_complete(self, prompt: str) -> AsyncGenerator[str, None]:
        """流式完成"""
        async with self.client.stream(
            "POST",
            f"{self.config.LLM_API_BASE}/chat/completions",
            headers={"Authorization": f"Bearer {self.config.LLM_API_KEY}"},
            json={
                "model": self.config.LLM_MODEL,
                "messages": [{"role": "user", "content": prompt}],
                "temperature": self.config.LLM_TEMPERATURE,
                "stream": True
            }
        ) as response:
            response.raise_for_status()
            async for line in response.aiter_lines():
                if line.startswith("data: "):
                    data = line[6:]
                    if data == "[DONE]":
                        break
                    try:
                        chunk = json.loads(data)
                        content = chunk["choices"][0]["delta"].get("content", "")
                        if content:
                            yield content
                    except json.JSONDecodeError:
                        continue
    
    async def close(self):
        await self.client.aclose()
```

### 3.6 API路由实现

```python
# api/routes/search.py
from fastapi import APIRouter, Depends, HTTPException
from fastapi.responses import StreamingResponse
from typing import AsyncGenerator
import json
import asyncio

router = APIRouter(prefix="/api/search", tags=["search"])

@router.post("/stream")
async def search_stream(
    doc_id: str,
    strategy: str,
    query: str,
    top_k: int = 3,
    chunk_service: AsyncChunkService = Depends(get_chunk_service),
    llm_service: AsyncLLMService = Depends(get_llm_service)
):
    """流式检索问答"""
    
    async def event_generator() -> AsyncGenerator[str, None]:
        try:
            # 1. 发送开始信号
            yield f"data: {json.dumps({'type': 'start', 'timestamp': asyncio.get_event_loop().time()})}\n\n"
            
            # 2. 检索chunks
            contexts = await chunk_service.retrieve_chunks(
                doc_id, strategy, query, top_k
            )
            
            yield f"data: {json.dumps({'type': 'contexts', 'contexts': contexts, 'count': len(contexts)})}\n\n"
            
            # 3. 构建prompt
            prompt = chunk_service.build_rag_prompt(contexts, query)
            
            # 4. 流式生成回答
            answer_chunks = []
            async for chunk in llm_service.stream_complete(prompt):
                answer_chunks.append(chunk)
                yield f"data: {json.dumps({'type': 'answer', 'content': chunk})}\n\n"
            
            # 5. 保存实验记录（异步，不阻塞响应）
            full_answer = "".join(answer_chunks)
            asyncio.create_task(
                chunk_service.save_experiment(doc_id, strategy, query, full_answer, contexts)
            )
            
            # 6. 完成信号
            yield f"data: {json.dumps({'type': 'done'})}\n\n"
            
        except Exception as e:
            yield f"data: {json.dumps({'type': 'error', 'error': str(e)})}\n\n"
    
    return StreamingResponse(
        event_generator(),
        media_type="text/event-stream",
        headers={
            "Cache-Control": "no-cache",
            "Connection": "keep-alive",
            "X-Accel-Buffering": "no"  # 禁用nginx缓冲
        }
    )

@router.post("/")
async def search(
    doc_id: str,
    strategy: str,
    query: str,
    top_k: int = 3,
    chunk_service: AsyncChunkService = Depends(get_chunk_service),
    llm_service: AsyncLLMService = Depends(get_llm_service)
):
    """非流式检索问答"""
    contexts = await chunk_service.retrieve_chunks(doc_id, strategy, query, top_k)
    prompt = chunk_service.build_rag_prompt(contexts, query)
    answer = await llm_service.complete(prompt)
    
    # 保存实验记录
    await chunk_service.save_experiment(doc_id, strategy, query, answer, contexts)
    
    return {
        "answer": answer,
        "contexts": contexts,
        "strategy": strategy
    }
```

```python
async def batch_create_chunks(
    self, 
    document_ids: List[str], 
    strategy: str,
    batch_size: int = 10
):
    """批量处理多个文档"""
    semaphore = asyncio.Semaphore(5)  # 限制并发数
    
    async def process_one(doc_id):
        async with semaphore:
            return await self.create_chunks(doc_id, strategy)
    
    tasks = [process_one(doc_id) for doc_id in document_ids]
    results = await asyncio.gather(*tasks, return_exceptions=True)
    return results
```

## 9. 常见问题与解决方案

### Q1: 同步代码和异步代码混用导致死锁？

**A**: 严格分离：异步层调用异步代码，同步CLI通过asyncio.run()调用异步代码，不要混用。

### Q2: pgvector向量检索在异步中如何实现？

**A**: 使用SQLAlchemy 1.4+的异步支持，向量操作通过text()执行原生SQL。

### Q3: 流式输出时如何保证实验记录一定保存？

**A**: 使用asyncio.create_task()后台保存，即使前端断开连接也继续保存。

### Q4: CLI命令执行时间长导致asyncio.run()超时？

**A**: 增加超时时间：`asyncio.run(coro(), timeout=300)`

## 10. 总结

### 改造收益

- ✅ CLI命令**100%保持可用**，用户体验不变
- ✅ API支持**高并发**（100+ QPS）
- ✅ 支持**流式输出**，可对接前端聊天
- ✅ 代码**向后兼容**，可渐进式迁移
- ✅ 原有业务逻辑**完全复用**

### 注意事项

- 异步改造需要**统一使用异步库**（httpx、asyncpg）
- 避免在异步代码中调用同步阻塞函数
- 合理设置**连接池大小**和**超时时间**
- 生产环境建议使用**gunicorn + uvicorn workers**
